LDAP management module for Hubot

Created a management module for LDAP, using Hubot robotics for the Slack messaging service. Code snippet can be found here, although requires the full repository to work (intellectual property not owned by me).
The code uses the Python LDAP module and GPG credentials for basic user management, e.g. creating and modifying users and groups.
Whilst many LDAP tools already exist, this solution suited us for a number of reasons:
• Slack provides a message history, allowing us to check tasks that had been done;
• Having this functionality in our messaging service made staff onboarding easier; and
• Using encrypted credentials allowed for a more secure environment, whilst allowing more staff members to perform basic tasks.

As a beginner at Python programming I found this project was very engaging, and it had a tangible and satisfying end result.
